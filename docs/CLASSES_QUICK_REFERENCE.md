# Classes & Profile Features - Quick Reference

## ğŸ¯ Quick Start

### For Students

#### Step 1: Update Your Profile
1. Click **"Profile"** tab in Student view
2. Upload a profile picture (optional)
3. Enter your information:
   - Full Name (required)
   - Student ID (required)
   - Email
   - School/Institution
   - Grade/Level
4. Click **"Save Profile"**

#### Step 2: Enroll in Classes
1. Click **"Classes"** tab in Student view
2. Enter the **class code** provided by your teacher
3. Click **"Enroll"**
4. You'll see all your enrolled classes

#### Step 3: Take Class Quizzes
1. Find your class in the "My Classes" section
2. Look at the "Class Quizzes" section
3. Click **"Take Quiz"** to start
4. Complete the quiz before the time runs out (if applicable)

---

### For Teachers

#### Step 1: Create a Class
1. Click **"Classes"** tab in Teacher view
2. Enter class details:
   - **Class Name** (e.g., "Physics 101")
   - **Subject** (e.g., "Physics")
   - **Grade** (e.g., "10th Grade")
   - **Description** (optional)
3. Click **"Create Class"**
4. **Share the class code** with your students

#### Step 2: Add Quizzes to Your Class
1. First, create quizzes in the **"Dashboard"** tab
2. Go back to **"Classes"** tab
3. Click **"+ Add Quiz"** on your class
4. Select a quiz from your list
5. Students will see it immediately!

#### Step 3: Manage Your Class
1. See how many students enrolled
2. View all quizzes in the class
3. Remove quizzes if needed
4. Delete the class when done

## âŒ Common Issues & Solutions

### "Invalid Class Code"
- **Problem**: Code doesn't exist or is wrong
- **Solution**: 
  - Check the code is correct (case-sensitive)
  - Ask teacher for the exact code
  - Verify no extra spaces

### "Already Enrolled"
- **Problem**: You're already in this class
- **Solution**: 
  - Check your "My Classes" section
  - You can only enroll once per class

### Profile Picture Won't Upload
- **Problem**: File too large or wrong format
- **Solution**:
  - Keep file under 5 MB
  - Use JPG, PNG, or GIF format
  - Compress image if needed

### Classes Not Showing
- **Problem**: Data not saved or lost
- **Solution**:
  - Clear browser cache and reload
  - Check localStorage quota
  - Create a new class/enrollment

### Timer Not Working
- **Problem**: Time limit not set
- **Solution**:
  - Teacher needs to set time limit when creating quiz
  - Quizzes without time limits have no timer

All your data is saved **locally in your browser**:

**Important**:

## ğŸ” Privacy & Security

### What's Stored Locally
- Your name and contact info
- Profile picture (base64 encoded)
- Classes you're in (students)
- Classes you created (teachers)
- Quiz responses and scores

### Google Account
- Optional connection
- Name and email stored
- Never used for unauthorized access
- Can disconnect anytime

### Who Can See Your Data
- **Only you** can see your profile
- **Your teachers** can see your enrollment
- **Your students** cannot see your profile
- **Your profile picture** shown in profile only

## ğŸš€ Tips & Tricks

### For Students
1. **Save your profile** - Don't lose it by clearing browser data
2. **Keep your class code** - Ask teacher to repeat if needed
3. **Check due dates** - Don't miss quiz deadlines
4. **Use your real name** - Helps teachers identify you
5. **Upload a profile picture** - Makes your profile complete

### For Teachers
1. **Share class code early** - Post on your syllabus
2. **Create quizzes first** - Then add to classes
3. **Monitor enrollment** - Check "My Classes" regularly
4. **Use meaningful names** - Include period/section
5. **Set time limits** - Creates more challenging quizzes
6. **Remove old quizzes** - Keep classes organized
